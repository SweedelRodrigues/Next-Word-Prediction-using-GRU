# 🧠 Next Word Prediction using GRU

This project demonstrates a Next Word Prediction system built using Gated Recurrent Unit (GRU) networks in TensorFlow and Keras. The model is trained on textual data to learn sequential patterns and contextual relationships between words, enabling it to predict the most likely next word for a given input sequence.

The workflow involves preprocessing text, tokenizing sentences, generating training sequences, padding variable-length inputs, and encoding target words. These processed sequences are then used to train a GRU-based neural network capable of understanding language dependencies and generating accurate next-word predictions.

This project provides a practical introduction to Natural Language Processing (NLP), sequence modeling, and recurrent neural networks, with applications in predictive text systems, chatbots, and text generation.

## ✨ Features
- Text preprocessing and tokenization
- Sequence generation and padding
- GRU-based language modeling
- Next word prediction from input sequences
- End-to-end NLP workflow implementation

## 🛠️ Tech Stack
- Python
- TensorFlow
- Keras
- NumPy
- Jupyter Notebook

## 📌 Workflow
Text Corpus → Tokenization → Sequence Generation → Padding → GRU Model Training → Next Word Prediction
